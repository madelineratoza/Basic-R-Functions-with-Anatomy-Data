# Basic R Functions with Anatomy Data

This repository contains a beginner-friendly introduction to using **R for basic data exploration**. The lesson walks through loading a dataset, examining variables, calculating descriptive statistics, creating summary tables, and producing a simple visualization.

The materials are designed for learners with **little or no prior coding experience**.

## Lesson Overview

This tutorial introduces several foundational R skills using a simulated dataset from an Anatomy course.

Topics covered include:

- Loading an Excel dataset into R
- Viewing variable names and data structure
- Calculating descriptive statistics
- Creating summary tables
- Comparing groups of students
- Producing a basic visualization using `ggplot2`

The lesson uses a simulated dataset that includes:

- pathway (Residential vs Hybrid)
- prior GPA
- final exam score
- NPTE score
- overall course score

## Learning Goals

After completing this lesson, learners should be able to:

- Load data into R from an Excel file
- Inspect the structure of a dataset
- Compute basic descriptive statistics
- Create summary tables using `dplyr`
- Visualize results with a simple bar chart
